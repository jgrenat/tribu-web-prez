## PWA - Les avantages

- Chargement ultra-rapide
- Gestion du mode Hors-ligne
- Beaucoup d'API disponnible (Notification Push, Vibration, Son, ...)
- Installation sur l'écran d'accueil
- 1 Développement = un site web et une app mobile
